To install the package, one of the following commands can be used:

!!! note "Package managers"

    === "npm"

        ```bash
        npm i joyrouter
        ```

    === "yarn"

        ```bash
        yarn add joyrouter
        ```

    === "pnpm"

        ```bash
        pnpm add joyrouter
        ```
